# Sample Dataset

This folder contains sample datasets used for demonstrating the analysis workflow.

The original datasets contain business-sensitive information and are therefore not included in this repository.

Files included:

- `sales_sample.xlsx`
- `purchase_sample.xlsx`
- `inventory_sample.csv`
- `expiry_sample.xlsx`

These sample datasets preserve the structure of the original data and allow the notebook to demonstrate the preprocessing and analysis workflow.
